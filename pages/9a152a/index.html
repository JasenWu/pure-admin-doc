项目打包

# 一：本地环境打包并预览

pnpm preview:build
1

# 二：正式环境打包

pnpm build
1

# 三： 预发布打包

  • 预发布 (opens new window) vite文档
pnpm build:staging
1

# 四：打包分析

pnpm report
1

# 五：项目文件、语言分析

pnpm cloc
1

注意

项目提供了全局打包路径VITE_PUBLIC_PATH，默认/，可根据需求自行修改
