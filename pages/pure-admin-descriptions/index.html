pure-admin-descriptions

# 简介

# 🤔 开发初衷

  • element-plus Descriptions (opens new window) 的Descriptions Item属性目前只能写在<template></template>模版里，这样不是很灵活，如果描述列表的Item足够多，代码写、看起来很臃肿，于是我开发了这个库，让我们一起探索吧

# 🚀 特性

  • 🦾 灵活度高: 使用tsx语法编写，保证类型的同时，给开发者提供了更灵活的写法，给使用者提供了更方便的配置
  • ⚡ 完全可摇树: 自带Tree-shaking，只对引入的代码进行打包
  • 🫶 代码零侵入: 保持element-plus Descriptions (opens new window) 所有属性、插槽的同时，提供更灵活的配置
  • ⚓ 代码提交前校验: 使用 husky (opens new window) 对提交代码前进行规则校验，强制规范开发流程，防止开发失误
